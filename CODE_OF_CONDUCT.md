Code of Conduct – Bozart Botanics

Bozart Botanics ist ein offenes Wissensprojekt.
Um eine freundliche, produktive und respektvolle Zusammenarbeit sicherzustellen, gilt dieser Verhaltenskodex.

1. Grundwerte

Wir verpflichten uns zu:

Respekt

Transparenz

Hilfsbereitschaft

Freundlichkeit

Wissenschaftlicher Genauigkeit

Verantwortungsvoller Sprache

2. Erwünschtes Verhalten

Respektvolle Kommunikation

Konstruktive Kritik

Präzise, faktenbasierte Beiträge

Einhaltung der Formatvorlagen

Bereitschaft zur Zusammenarbeit

Geduld und Höflichkeit im Austausch

3. Unerwünschtes Verhalten

Folgendes wird nicht toleriert:

Belästigung, Beleidigungen, aggressive Sprache

Rassistische, sexistische, homophobe oder sonst diskriminierende Inhalte

Spam, Werbung oder externe Verkaufslinks

Verbreitung medizinischer Falschinformationen

Verleumdung, Gerüchte oder pseudowissenschaftliche Aussagen

Manipulierte Daten oder irreführende Behauptungen

Untergrabung der Projektstruktur

4. Umgang mit Verstößen

Bei Verstößen behält sich das Projektteam vor:

Verwarnungen auszusprechen

Beiträge abzulehnen oder zu entfernen

Benutzer aus Diskussionen auszuschließen

Benutzer dauerhaft zu sperren (bei schwerwiegenden Fällen)

5. Melden eines Problems

Probleme oder Verstöße können gemeldet werden über:

GitHub Issues

Pull Request Kommentare

Bitte liefert eine sachliche Beschreibung und ggf. Beispiele.

6. Geltungsbereich

Dieser Kodex gilt für:

alle Beiträge im Repository

Pull Requests

Issues

Diskussionen in Kommentaren

Interaktionen über verknüpfte Plattformen

7. Schlusswort

Bozart Botanics lebt von der Idee, Wissen frei, korrekt und respektvoll zu teilen.
Dieser Kodex stellt sicher, dass die Community sicher, professionell und wertschätzend bleibt.
