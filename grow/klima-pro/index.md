# Fortgeschrittene Klimasteuerung  
CO₂ · VPD · Temp · RLF – Das High-End Indoor-Klima

Klimasteuerung ist der Schlüssel für:

- Wachstumsgeschwindigkeit  
- Ertrag  
- Photosyntheseleistung  
- Transpiration & Nährstofftransport  
- Terpenproduktion  
- Stressresistenz  
- CO₂-Effizienz  

Ein perfekter Düngerplan bringt nichts, wenn das Klima instabil ist.

Dieser Guide erklärt die **komplette Profi-Klimaführung**, wie sie in Indoor-Farmen, RDWC-Systemen und CO₂-optimierten Zelten eingesetzt wird.

---

# 1. Warum Klima wichtiger ist als Dünger

Nährstoffaufnahme entsteht nur durch **Transpiration**.  
Transpiration wird gesteuert durch:

- Temperatur  
- Luftfeuchtigkeit (RLF)  
- VPD  
- Luftstrom  
- CO₂  

Wenn Klima falsch ist:

- EC steigt  
- Mängel erscheinen  
- Wachstum stoppt  
- Blüten bleiben klein  
- Pflanzen hängen / „canoeing“  
- Wurzeln werden schwach

→ **Klimafehler = 80 % aller Grow-Probleme**

---

# 2. VPD – der wichtigste Wert im gesamten Grow

VPD = Vapor Pressure Deficit  
→ misst den Druckunterschied zwischen Blatt & Umgebung.

Er steuert:

- Wasseraufnahme  
- Nährstofffluss  
- Transpiration  
- Blatttemperatur  
- Stresslevel  

### Optimale VPD-Werte:

| Phase | VPD |
|-------|------|
| Keimung | 0.4–0.7 |
| Sämlinge | 0.7–0.9 |
| Veg | 0.9–1.2 |
| Frühe Blüte | 1.1–1.3 |
| Vollblüte | 1.3–1.5 |
| Späte Blüte | 1.4–1.6 |

**Zu niedriger VPD → zu feucht**  
→ Pflanze transpiriert schlecht, Nährstofffluss minimal

**Zu hoher VPD → zu trocken**  
→ Blattstress, erhöhter EC, verbrannte Spitzen

---

# 3. Temperatur – die echte Photobiologie

### Tag/Beleuchtung:
- **24–28°C** (optimal für LED)
- mit CO₂ **28–30°C**

### Nacht:
- **18–22°C**

### Unterschied Tag/Nacht:
- nicht mehr als **6°C**  
→ stabilisiert Terpene  
→ verhindert Stretch

---

# 4. RLF – relative Luftfeuchtigkeit

### Phasen-Richtwerte:

| Phase | RLF |
|-------|------|
| Keimung | 70–80 % |
| Veg | 55–70 % |
| Stretch | 50–65 % |
| Blüte | 45–55 % |
| Späte Blüte | 40–50 % |
| Letzte 3 Tage | 35–45 % |

**Hohe RLF in Blüte → Botrytis-Risiko**  
**Niedrige RLF in Veg → Spinnmilben-Risiko**

---

# 5. CO₂ – High-End Klimaparameter

CO₂ steigert:

- Photosynthese  
- Wachstumsgeschwindigkeit  
- Blütendichte  
- Ertrag  
- Resistenz gegen Stress  

Aber:

**CO₂ funktioniert nur, wenn PPFD + DLI hoch genug sind.**

### Empfohlene CO₂-Werte:

| CO₂ | PPFD benötigt | Effekt |
|-----|---------------|--------|
| 400 ppm | 400–800 | Standard |
| 800 ppm | 800–1000 | +20–30 % Wachstum |
| 1200–1500 ppm | 1000–1200 | Profi-Level |

CO₂ erhöht den optimalen Temperaturbereich:

- ohne CO₂: **24–28°C**  
- mit CO₂: **28–30°C**

---

# 6. Wie Klima + CO₂ + Licht zusammenhängen

### Die 3 wichtigsten Kopplungen:

1. **PPFD ↑ → CO₂ ↑**  
2. **CO₂ ↑ → Temperatur ↑**  
3. **Temperatur ↑ → VPD neu einstellen**

Beispiel:

Wenn du PPFD auf 900 µmol erhöhst:

- CO₂ auf 800–1000 ppm  
- Temperatur auf 27–29°C  
- RLF so einstellen, dass VPD 1.1–1.3 ist

→ Das ist die perfekte „High-Performance-Zone“.

---

# 7. Luftstrom – der unsichtbare Klimaregler

Luftbewegung beeinflusst:

- Blatt-Temperatur  
- Gas-Austausch  
- CO₂-Schicht an Blättern  
- Schimmelrisiko  
- Stammeinlagerung („Windhärte“)

### Regeln für perfekten Luftstrom:

- Zwei Clip-Fans  
- Einer oben, einer unten  
- Kein direkter Wind auf die Buds → Windbrand  
- Leichte, gleichmäßige Wellenbewegung

---

# 8. Temperatur-Blatt vs. Temperatur-Luft

Blatt-Temperatur ist **wichtiger als Luft-Temperatur**.

LEDs erzeugen weniger IR → Blätter kühler.

### Typischer Unterschied:
- Blatt 2–3°C kühler als Luft

Wenn Luft 26°C → Blatt 23–24°C

Daher:

- Lufttemperatur um **2–3°C höher fahren**  
damit die Blatt-Temperatur passt.

# 10. Tools für perfekte Klimasteuerung

Empfohlen:

- **Inkbird Temp & RLF Controller**  
- **Govee oder Genius Hub Sensoren**  
- **Entfeuchter**  
- **Luftbefeuchter**  
- **EC/PH-Live-Meter (Hydro)**  
- **CO₂-Messgerät / Controller**  
- **Oscillating Fans**

---

# 11. Klima-Fehlerdiagnose (Tabelle)

| Symptom | Ursache | Lösung |
|--------|---------|--------|
| Blätter hängen | VPD zu hoch oder zu niedrig | Klima anpassen |
| gelbe Ränder | Hitze + zu hohe PPFD | Lampe höher/dimmen |
| verbrannte Spitzen | VPD zu hoch → EC steigt | RLF erhöhen |
| Mehltau | RLF zu hoch | Entfeuchtung |
| Botrytis | schlechte Luftbewegung | Fans ändern |
| Stretch zu krass | Temp zu hoch, VPD zu niedrig | Klima anpassen |
| matte Blätter | CO₂ zu niedrig | CO₂ erhöhen |

---

# 12. High-End Klimataktiken (Profis)

### A) „Cool Nights Strategy“
- Nacht 3°C kälter  
→ fördert Purple-Farben (Anthocyane)

### B) „High CO₂ Burst“
- 1200–1500 ppm für 1–3 Stunden während Peak-PPFD  
→ maximale Photosynthese

### C) „Low RH Final Week“
- 35–40 % RLF  
→ härtere Buds, weniger Botrytis

### D) „Heat + CO₂ Power Mode“
- 29–30°C + 1200 ppm  
→ maximale Blütendichte (nur für Profis)

---

# 13. Fazit

Perfektes Klima bedeutet:

- massive Erträge  
- schnelle Growzyklen  
- maximales Aroma  
- starke Terpenprofile  
- stabile Nährstoffaufnahme  
- keine Krankheiten  

Klimasteuerung ist die **Meisterdisziplin jedes Growers**.

Je besser Klima → desto weniger Nährstoffprobleme → desto mehr Erfolg.
